# Hi_JustFast

*有些人，住在世界的褶皱里。*  
*我们想去那里，带上一点光。*

---

我们是一个刚刚起步的开源小组。  
没有宏大的计划，没有完美的代码，只有一个朴素的念头——

**用AI和硬件，去靠近那些容易被忽视的人。**

养老院里独坐的老人，山村里看不起病的孩子，  
街头听不见红绿灯的盲人，工厂里疲惫的流水线工人……  
只要是与民生有关的困境，我们都愿意试着做点什么。

---

## 我们能做什么

技术方向大致落在这里，但边界是模糊的，  
因为真实的需求从来不按类别生长：

- 🤖 边缘 AI 推理 · 让算力贴近现场
- 📡 低功耗传感网络 · 悄悄守护，不打扰
- 👁️ 计算机视觉 · 看见那些被忽略的细节
- 🗣️ 语音与自然交互 · 让不懂科技的人也能用
- 🔧 开源硬件 · 低成本，可复制，可修改

---

## 我们的现状

坦白说，我们刚刚成立，一切都还很粗糙。

代码在生长，想法比代码跑得快，  
有些东西还没想清楚，有些项目才刚种下。

如果你来了，看见这里乱糟糟的，请多包涵——  
但如果你愿意留下来，我们会一起把它变得更好。
